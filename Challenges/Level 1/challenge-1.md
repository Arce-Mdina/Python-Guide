**Challenge:** Building a program where it adds two entered numbers together.

**Skills required:** Basic input & output; variables; 

**Possible Solution**:
```python
# First number input
print("Enter a number:")
num_1 = int(input(">"))

# Second number input
print("Enter another number:")
num_2 = int(input(">"))

# Final answer
answer = num_1 + num_2
print(num_1, "+", num_2, "=", answer)
```
